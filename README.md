# Smart Farming Frontend

A React-based dashboard for smart farming operations, featuring sensor data visualization, crop monitoring, weather integration, and irrigation control.

## Features

- Real-time sensor data display (temperature, humidity, soil moisture)
- Crop monitoring dashboard
- Weather widget
- Irrigation control system

## Getting Started

1. Install dependencies:
   ```
   npm install
   ```

2. Start the development server:
   ```
   npm run dev
   ```

3. Open your browser to `http://localhost:5173`

## Build

To build for production:
```
npm run build
```

## Technologies Used

- React
